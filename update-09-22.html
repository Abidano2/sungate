SUNGATE - 2022 Progress

A whole year has passed since the start of this game yet it feels like it's only been a few months...

As you can see, the game has its very own website now! It doesn't have much content right now but it'll definitely be used more in the future.

For the people who have no clue what this is since I've only just started to share it more, SUNGATE is a Japanese-style RPG inspired by games such as Deltarune, so if you're into those kind of games I guess this is for you? I can't really talk about what the game is about as of now so pretend like it's some sort of surprise (I can't 100% promise it will be a good one though as this is my first full game...).

Now, this introduction has been going on for too long so why don't we talk about progress more?

Development

In terms of development, most of it was focused on design, specifically story-writing, for the past year. Some parts of the beginning were implemented including a dialogue system however I decided to restart all of it as they were very messy and the systems didn't work properly. So only the chapter select screen has been fully completed so far.

In terms of the story, most of it is complete. Main characters/bosses are all known, general progression is known, main locations are know and so on. If I get the time, I might finish it all completely by the end of this year!
